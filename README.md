## Component

<br>

## 개발환경 구축하기
- 설치 <br>
`npm install -g create-react-app`
- 설치 확인 <br>
`create-react-app -V`
- 프로젝트 생성하기
  1. 빈 폴더를 만든다
  2. 폴더의 경로로 들어가 `create-react-app .` 입력
  3. 프로젝트 생성 완료

<br>

## 실행하기
`npm run start`

<br>

## npm 과 npx의 차이
- npm이 그 프로그램을 설치하는 거라면
- npx는 create-react-app을 한 번만 설치한다.
  - 메모리를 차지하지 않는다.
  - 다운로드 받을 때마다 최신 상태이다.

<br>
